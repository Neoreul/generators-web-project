# generators-web-project

##Download
```
git clone https://github.com/tanUIT/generators-web-project.git
```
##Installation
```
npm install
```
##Usage
- To run project with browser-sync use:
```
gulp serve
```
- To build project as production use:
```
gulp
```
- To minify css, js, images use:
```
gulp compress
```
